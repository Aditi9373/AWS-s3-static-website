# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates hosting a static website on **Amazon S3** using static website hosting.

## AWS Services Used
- Amazon S3
- IAM (basic access management)

## Features
- Static website hosting using S3
- Public access using bucket policy
- Custom error page configuration

## Steps Implemented
1. Created an S3 bucket
2. Uploaded static website files (HTML, CSS)
3. Enabled static website hosting
4. Configured index and error documents
5. Applied bucket policy for public access
